Andere Gruppe {% if player.id_in_group > 3 %} "Blau" {% else %} "Grün" {% endif %}

Jedes Mitglied der Gruppe {% if player.id_in_group > 3 %} "Blau" {% else %} "Grün" {% endif %} kann ebenso in 2 verschiedene Töpfe (A und B) einzahlen oder das Guthaben behalten. Für jeden Euro in Topf A erhält jedes Mitglied der anderen Gruppe 0,50 €. Für jeden Euro in Topf B erhält jedes Mitglied der anderen Gruppe ebenfalls 0,50 €. Zusätzlich werden aber jedem Mitglied Ihrer Gruppe 0,50 € abgezogen.

Fall 1

Nehmen Sie nun an, jedes Mitglied der {% if player.id_in_group > 3 %} "Blau" {% else %} "Grün" {% endif %} hätte 5 € in Topf A eingezahlt, 0 € in Topf B und 5 € behalten.

Für jeden Euro in Topf B der Gruppe {% if player.id_in_group > 3 %} "Blau" {% else %} "Grün" {% endif %} werden Ihnen und jedem Mitglied Ihrer eigenen Gruppe {% if player.id_in_group < 4 %} "Blau" {% else %} "Grün" {% endif %} nun 0,50 € von der Endauszahlung abgezogen.

Im Topf A sind 15 €. In Topf B sind 0€. Somit werden Ihnen 0 € von 12,50 € abgezogen.
